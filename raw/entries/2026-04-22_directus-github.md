# directus/directus

Источник: https://github.com/directus/directus

Ключевые факты из репозитория и описания:
- Directus позиционируется как real-time API и App dashboard поверх SQL database content.
- Даёт REST и GraphQL API поверх существующей или новой SQL-базы.
- Поддерживает PostgreSQL, MySQL, SQLite, OracleDB, CockroachDB, MariaDB и MS-SQL.
- Работает с чистой SQL-схемой без обязательной миграции в проприетарный формат.
- Включает no-code dashboard для управления данными.
- Поддерживает self-hosted и cloud-модель развёртывания.
- Платформа расширяема и подходит для headless CMS, admin panels и app backends.
