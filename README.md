# Wiki Chiki — LLM-Powered Knowledge Base

Автоматическая wiki по инфраструктуре, менеджменту и LLM/агентам.

## Структура

- `raw/` — исходные документы (immutable)
- `raw/entries/` — обработанные markdown-файлы
- `wiki/` — скомпилированные статьи
- `SCHEMA.md` — инструкции для LLM
- `log.md` — журнал операций

## Как использовать

1. Дай LLM источник (URL, файл, текст)
2. LLM обработает его и обновит wiki
3. Задавай вопросы — LLM найдёт ответ по статьям

## Предметные области

- **Инфраструктура** — сети, серверы, виртуализация, мониторинг, IaC
- **Менеджмент** — процессы, команды, ITSM, коммуникации
- **LLM и агенты** — модели, промпты, RAG, MCP, деплой
