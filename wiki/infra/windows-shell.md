# Windows Shell

Графическая оболочка Windows — совокупность процессов, отвечающих за рабочий стол, панель задач, меню Пуск, системный трей и оформление окон.

## Компоненты в Windows 10/11

| Процесс | Назначение | Типичное потребление |
|---------|-----------|---------------------|
| explorer.exe | Рабочий стол, файловый менеджент | 50-150 МБ |
| StartMenuExperienceHost.exe | Меню Пуск (XAML) | 30-80 МБ |
| ShellExperienceHost.exe | Оформление, прозрачность | 20-60 МБ |
| SearchHost.exe | Поиск в меню Пуск | 30-50 МБ |
| RuntimeBroker.exe | UWP-инфраструктура | 10-30 МБ |

Итого: 140-370 МБ только на shell, без учёта фоновых UWP-процессов.

## Альтернативные оболочки

Shell Windows можно заменить целиком или модульно:

- **RetroBar** — замена панели задач (Win32 API, .NET)
- **OpenShell / Retro-shell** — замена меню Пуск
- **Float** — альтернативный WM для Windows
- **BB4Win / LiteStep** — полностью кастомные shell (старые)

Модульный подход (замена отдельных компонентов) проще в поддержке и стабильнее, чем полная замена explorer.exe.
