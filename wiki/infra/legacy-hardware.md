# Легаси-железо

Старое оборудование, которое ещё выполняет свои функции, но не справляется с современными программными требованиями.

## Контекст

Распространённая ситуация в корпоративной и домашней среде: пользователь отказывается менять привычную машину. Причины — устоявшиеся рабочие процессы, «намоленные» программы, нежелание переучиваться.

## Проблема

Современные ОС и приложения растут в требованиях. Windows 10/11, современный браузер, офисный пакет — на 2-4 ГБ ОЗУ это страдание. UI один отъедает 300-500 МБ.

## Стратегии продления жизни

1. **Оптимизация софта** — замена тяжёлых компонентов на легковесные (см. [Оптимизация рабочего стола Windows](/wiki/infra/windows-desktop-optimization/))
2. **Легковесные ОС** — Linux-дистрибутивы (Lubuntu, Xubuntu, Tiny Core)
3. **Аппаратные апгрейды** — SSD вместо HDD, добавление ОЗУ (если поддерживается)
4. **VDI/тонкие клиенты** — перенос вычислений на сервер

## Когда не стоит спасать

- Нет обновлений безопасности
- Аппаратные ограничения не позволяют запустить необходимый софт
- Стоимость продления жизни сравнима с заменой
